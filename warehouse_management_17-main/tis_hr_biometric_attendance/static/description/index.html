Biometric Attendance

This module helps to integrate Odoo HR Attendance with ZK attendance machine amd it supports Zkteco Device such as F18, uFace202, F19, iClock 680, MA300, F16, F22, uFace800, SpeedFace-V4L, F28, MB10, etc.

(This doesn't support on Odoo.sh)

1. Device Configuration

Navigate to the Attendance module and click on Biometric Device Config menu to configure the device.

2. Enter the device details.

Configure a static Device IP, Port and Timezone for the system. If the biometric device has password, you can configure it by enabling Is Password Set field. After configuration click the Test Connection button.

3. Success Message.

If the connection is success, server shows a success message as shown.

4. Error Message.

If the connection is not success, server shows a error message as shown.

5. Synchronize Employees with the Biometric Device. If you are able to sync the employees with the device, you can see a popup with a success message.

Click the Sync Employees smart tab to synchronize employees with biometric device.

If the synchronize employees is success, server shows a success message as shown.

6. Action Menus

Navigate to the Employee form and you can see three menus in employee form.

7. Scan with Biometric

Click on Scan with Biometric, to scan employee with the biometric device. In the popup select the biometric device and click on scan

8. Update with Biometric

Click on Update with Biometric, to update the employee details in the biometric device. In the popup select the biometric device and click on update

9. Biometric Devices tab.

After updating the employee details, a line will be created under the Biometric Devices tab. This line maps the same ID present on the biometric device to the Biometric User Id field.

10. Remove from Biometric.

Click on Remove with Biometric, to remove the employee details from the biometric device. In the popup select the biometric device and click on remove

11. Card Number.

The Card Number field is now available under the HR Settings tab.If you need assign a card number to an employee, this card number will be passed to the biometric device when you click the Sync Employees button in the Biometric Configuration form.

12. Download Attendance Logs

Click the Download Attendance smart tab to download the attendance log.

If downloaded successfully,then it shows a success message.

13. Biometric Attendance Log.

Click on Biometric Attendance Log to see the downloaded attendance logs.

14. Calculate Attendance.

Under the Manger menu click on Calculate Attendance to calculate the attendance and from the popup click on Calculate.

After that a popup will show and from the popup click on Calculate.

15. Minimal Attendance.

To calculate attendance for each day only, enable 'Minimal Attendance' in configuration settings, and then click on the Calculate smart tab from Attendance moudule.

16. Overview menu.

From the Overview menu to see the employees attendances.

17. Attendance Report.

By clicking Attendance Report menu,

After that you can see a popup. Select the report type , from date and To date from here.
